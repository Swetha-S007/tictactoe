# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game made in Python where the player (**X**) plays against the computer (**O**).

## Features

* Single-player mode
* Random computer moves
* Win detection
* Tie detection

## How to Run

```bash
python tic_tac_toe.py
```

## Board Positions

```text
1 | 2 | 3
---------
4 | 5 | 6
---------
7 | 8 | 9
```

Enter a number from **1 to 9** to place your mark.

## Requirements

* Python 3.x
